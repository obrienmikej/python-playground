# Overview

Repo to use python.

## Environment

- Visual Studio Code
- Mac

##  Requirements

- Create a virtual environment and activate it to use the repo within VSCode.

- Install the dependencies via the `requirements.txt` file using.

```commandline
pip install -r requirements.txt
```

# Pre-Reqs for CDK

- Setup to use CDKTF with this repo.

### Terraform
```commandline
terraform --version
brew tap hashicorp/tap
brew install hashicorp/tap/terraform
```

### Node
```commandline
node --version
brew install node
```

### Yarn
```commandline
yarn --version
brew install yarn
```

### CDKTF
```commandline
cdktf --version
brew install cdktf
```
